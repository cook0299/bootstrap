Bootstrap

Installation Of Bootstrap

Looking to quickly add Bootstrap to your project? Use BootstrapCDN, provided for free by the folks at StackPath. All you have to do to get Bootstrap into your project, is go to bootstrap.com and proceed to the "Get Started" button, then scroll down to the "starter template" and copy paste that code into your current HTML. Boom, easy as that.

What Is So good About It?

Bootstrap is a great choice for making a responsive website. bootstrap has a great fluid grid system and responsive utility classes. This makes creating a responsive website a smooth and easy task. Bootstrap prioritizes mobile view to make different media screen sizes see the website in the most interactive way.

Options Are Endless

Bootstrap makes everything you do to your website a little bit easier. Customizing a website can be done with something as easy as going on bootstraps website and searching what you want to add... For example: having a 3 column text layout can be done as easy as putting a div with the class "col-md-4".
